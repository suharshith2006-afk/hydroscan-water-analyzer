# Smart Water Quality Analysis Prototype 🌊🧪

A conceptual mobile-based water quality detection prototype system engineered to simulate liquid spectrometry and chemical analysis. This application models Beer-Lambert law calculations, parsing relative liquid absorbance parameters and heavy metal contamination metrics using a smartphone camera and flashlight as a native optical light source.

🌐 Explore the hydroscan water analyzer : https://hydroscan-water-analyzer.netlify.app/

---

## 📌 Project Overview

This framework translates programmatic theories into practical applications by allowing real-time estimation of liquid absorbance parameters. It features a clean, responsive scanning dashboard optimized for mobile browsers to facilitate smooth on-device presentations.

### Key Features
* **Optical Sensing Interface:** A structured scanning layout designed to simulate sample testing using a smartphone camera and flashlight.
* **Spectrometry Modeling:** Integrated algorithmic engine to compute relative liquid absorbance and transmission variables.
* **Responsive Layout:** Engineered with strict UI design parameters using CSS Grid and Flexbox for seamless cross-device compatibility.
* **Clean Code Architecture:** Developed using semantic, lightweight scripting for efficient browser runtime execution.
